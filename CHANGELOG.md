# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-01-21

### Mobile Optimization
- **UI**: 针对移动端优化了标题区域的显示逻辑
  - **缩小字号**：标题从超大字号调整为适配手机的紧凑字号
  - **隐藏干扰项**：移动端自动隐藏"展开数量/显示节点"等非必要统计信息
  - **布局调整**：移除了顶部过多留白，避免遮挡画布核心区域

### Fixed
- **Deployment**: 修复了 GitHub Pages 静态导出的配置问题 (Added generateStaticParams)

---

## [1.0.0] - 2026-01-21

### Milestone
- **正式版发布**：完成核心交互与视觉优化
  - **完美视图逻辑**：解决缩放漂移、最大化问题，实现无缝的上下文切换
  - **动态物理引擎**：基于拓扑结构的自适应斥力系统 (Aggressive Tuning)
  - **高性能交互**：无重绘的平滑展开与稳定布局

### Fixed
- 修复切换主节点时画面中心偏移的问题（实施物理坐标重置）
- 修复小规模图谱（如边缘人物）导致视图过度放大的问题 (MaxZoom constraint)

---

## [0.1.2] - 2026-01-21

### Changed
- **物理引擎调优 (Physics Optimization)**
  - 实施动态边长策略：主节点与展开节点的连线显著延长 (550px)，将二级人物簇推向外层
  - 增强全局互斥力：电荷强度提升至 -2000，作用范围扩大至 3000px，杜绝节点重叠
  - 降低中心引力：允许图谱自然扩散，提升阅读清晰度

## [0.1.1] - 2026-01-21

### Fixed

  - 修复展开节点时视图意外重置（最大化）的问题

## [0.1.0] - 2026-01-21

### Added
- **可拓展的人物交互系统**
  - 支持通过右键/点击菜单“展开节点”，动态显示关联人物
  - 核心节点（Main Character）始终保持可见
  - 支持“设为主节点”功能，重新以该人物为中心构建图谱
- **层级化视觉样式 (Hierarchical Styling)**
  - **一级连线**（主节点连接）：**琥珀色 (Amber)** 高亮，线条加粗
  - **二级连线**（次级节点展开）：**蓝色 (Blue)** 高亮
  - **默认连线**：深灰色，清晰可见
- **渐进式图谱展示**
  - 初始仅显示核心人物网络，避免信息过载
  - 点击展开后动态添加节点，保持布局稳定（无刷新）
- **操作优化**
  - 新增“重置”按钮，一键恢复初始状态
  - 优化移动端点击与桌面端右键的统一交互体验

### Changed
- **底层重构**：从全量刷新改为增量更新 Graph 数据，保留节点位置
- **样式架构**：采用 CSS Class 强样式策略，解决样式覆盖问题
